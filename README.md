# Customized E-commerce Platform

A frontend-based customized e-commerce website that allows users to browse, customize, and order personalized products such as T-shirts, mugs, photo frames, wallets, and more.  
This project is developed as an **academic project** using **HTML and CSS**, focusing on UI/UX design and user flow.
